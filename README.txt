README – Site vitrine d’entreprise
Description du projet

Dans le cadre de notre apprentissage du JavaScript et du développement web, nous avons réalisé un site vitrine pour l’entreprise des parents de Lohan Darques.

Leur ancien site étant devenu obsolète, l’objectif était de proposer une version plus moderne, esthétique et fonctionnelle, afin de mieux représenter l’entreprise ainsi que ses différents services.

Ce projet nous a permis de mettre en pratique nos compétences en HTML, CSS et JavaScript tout en répondant à un besoin réel et concret.

Objectifs du projet

Ce projet devait répondre aux critères de l’Open Project :

- utilisation efficace de JavaScript
- amélioration de l’expérience utilisateur grâce à l’interactivité
- originalité et pertinence du projet
- qualité du code et respect des bonnes pratiques
- bon fonctionnement général et performances optimisées

Notre choix s’est porté sur la refonte complète d’un ancien site existant afin de proposer une solution plus moderne et professionnelle.

Fonctionnalités principales

Le site comprend plusieurs pages permettant de présenter l’entreprise et ses activités :

page d’accueil
page calcul et conception
page fabrication et pose
page réalisations
page export
page contact

L’objectif principal était de rendre la navigation simple, fluide et agréable pour l’utilisateur.

Fonctionnalités principales : 

Caroussel d'images : système de slider automatique présent sur les pages calcul, fabrication.

Slider manuel : Slider d'images d'illustration présent sur la page index

Carte interactive : Rend la page export.html interactive en permettant à l'utilisateur de se déplacer sur la page en cliquant sur les différents éléments présents sur la carte


Innovation et Originalité : 

Notre projet se distingue par son aspect concret. En effet, ici nous étions confronté à un besoin réel : moderniser le site d’une entreprise existante.

Cela nous a permis de travailler dans une logique plus professionnelle, en tenant compte des attentes d’un véritable client.

Le projet possède donc une réelle utilité pratique.

Lancer le projet : 

Le projet est disponible via l'URL suivant : https://github.com/ThomasDebuigne/PROJETBENAULT

Il suffit de : 

Cloner le dépôt GitHub
Ouvrir le dossier du projet
Lancer le fichier index.html sur un navigateur web
